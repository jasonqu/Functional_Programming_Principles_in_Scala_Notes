Nested functions

对sqrt来说，sqrtIter, improve和isGoodEnough都是其内部函数，不需要被外界看到

可以使用内部嵌套函数的做法来实现，避免命名空间泛滥"name-space pollution"

The sqrt Function, Take 2


  def sqrt(x: Double) = {
    def sqrtIter(guess: Double, x: Double): Double =
      if (isGoodEnough(guess, x)) guess
      else sqrtIter(improve(guess, x), x)
    def improve(guess: Double, x: Double) =
      (guess + x / guess) / 2
    def isGoodEnough(guess: Double, x: Double) =
      Math.abs(guess * guess - x) < 0.001
    sqrtIter(1.0, x)
  }

Blocks in Scala

包含一系列表达式，最后一个表达式代表其(返回)值

block本身也是表达式，可以出现在任何表达式出现的地方

Lexical Scoping refactor

block本身有变量的表达范围，之前的方法可以重构：


  def sqrt(x: Double) = {
    def sqrtIter(guess: Double): Double =
      if (isGoodEnough(guess)) guess
      else sqrtIter(improve(guess))
    def improve(guess: Double) =
      (guess + x / guess) / 2
    def isGoodEnough(guess: Double) =
      Math.abs(guess * guess - x) < 0.001
    sqrtIter(1.0)
  }

Semicolons and infix operators

scala中，;是可选的，简化代码，不过存在以下问题


someLongExpression
+ someOtherExpression

会被理解为


someLongExpression;
+ someOtherExpression

Semicolons and infix operators

解决方式：


(someLongExpression
+ someOtherExpression)

或


someLongExpression +
someOtherExpression
