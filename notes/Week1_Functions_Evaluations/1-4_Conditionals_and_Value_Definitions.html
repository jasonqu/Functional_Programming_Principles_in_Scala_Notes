Conditional Expressions

和Java的if-else类似，但是不同于java用于指令(statement)，scala中用于表达式(expressions)

def abs(x: Int) = if (x >= 0) x else -x

Boolean Expressions

Question: 找到一个在 CBN 下终止，但在CBV下不终止的表达式.

  • true false // Constants
  • !b // Negation
  • b && b // Conjunction
  • b || b // Disjunction

Rewrite rules for Booleans

布尔表达式的推到规则

  • !true --> false
  • !false --> true
  • true && e --> e
  • false && e --> false
  • true || e --> true
  • false || e --> e

Value Definitions

之前看到函数定义中存在“按值”和“按名”的区别，在变量定义中也有相同的区别

val是by value, 而def是by name

对val，是在定义时取值，如

  • val x = 2
  • val y = square(x)

y在定义时取值，为4

Value Definitions and Termination

查看这两种方式的不同：

def loop: Boolean = loop

  • def x = loop 是可以的
  • val x = loop 则会陷入死循环
